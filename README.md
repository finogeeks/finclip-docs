## FinClip 是什么?
[FinClip](https://www.finclip.com/)  是基于自研、领先的小程序容器技术，让任何企业的 APP 均获得嵌入该容器而瞬间运行小程序的能力，轻松实现移动应用的动态发布，与业务内容的跨端投放。

FinClip提供私有化解决方案，本项目旨在提供FinClip私有化的运行维护的文档支持，由FinClip运维团队提供。当然，如果您有意与我们一同完善这个文档项目，也欢迎提交内容到本仓库。
