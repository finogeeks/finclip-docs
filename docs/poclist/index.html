一. 前言#

  POC测试，即Proof of Concept，是业界流行的针对客户具体应用的验证性测试，通常是企业进行产品选型时或开展外部实施项目前，进行的一种产品或供应商能力验证工作。其目的是证明企业选择的产品或供应商能够满足需求，并且提供的信息准确可靠。


  本手册为FinClip小程序开放平台POC测试的指导手册，旨在为企业开发者提供FinClip产品的POC指引，通过本文档，您可以快速的**将小程序运行在您的移动应用App**上。 ## 二. POC 验证功能清单 通过本指导手册，您可以验证的功能清单如下：

五. POC流程概览#

本手册以将小程序运行在您的移动应用App上为目标，一共需要4个步骤和17个操作：
  1. 小程序开发调试：将小程序在FinClip IDE中编译通过，并做好相关的代码调试和布局，确保能够在模拟器上正常展示。
  2. 发布小程序：在FinClip小程序开放平台注册账号，登录平台，通过小程序模块创建小程序，填写小程序信息。步骤一调试好的小程序，提交小程序审核您将得到一个小程序 AppId ，后续可通过AppId打开小程序。
  3. 创建应用，关联小程序：在管理后台添加移动应用，关联小程序，此步骤您将会的获得SDK Key . SDK secret. apiServer三个参数，后续可用于初始化SDK。
  4. 集成SDK，打开小程序：移动应用集成SDK，采用步骤五的信息进行SDK初始化，调用SDK打开小程序接口，将步骤4中的AppId传入，即可在App中打开小程序。
  至此，app打开小程序步骤完成，下面是整个流程的时序图包含17个操作：

六. POC流程#

步骤一 注册开发者账号#

  如果您采用的是凡泰公有云方案，可以通过浏览器打开页面 ：注册地址 ，创建账号，即可完成账号的注册。
  如果您采用的是私有云的方案，您需要在您的私有云环境，创建一个账号。账号注册流程参考文档：平台注册及认证，完成第1个步骤即可。

步骤二 创建小程序#

  登录步骤一注册的账号后，您就可以进入到小程序开放平台管理后台，您可以通过左侧的菜单栏查看到整个开放平台的功能。
  进入小程序管理，小程管理-我的小程序，点击页面上角新增小程序按钮，填写小程序的名称、 小程序分类、等信息，即可完成小程序的创建。
  通过此步骤，您将得到小程序的id，即AppId。   创建小程序参考文档：创建小程序。

注：如果您需要升级为企业账号，但是受限于企业的营业执照授权书等信息，您可以先上传测试图片，后续通过修改操作变更为真实信息，提交审核后联系交付经理处理

步骤三 开发编译小程序#

  打开凡泰FIDE代码编辑器，登录步骤一注册的账号，通过FIDE创建一个小程序或者是打开现有的小程序。
  如果您是通过Taro. uni-app框架开发的，请先转成微信小程序，再使用FIDE打开小程序路径下的小程序源代码。
  如果您是是H5 的应用，想要内嵌到小程序的，可以使用web-view组件来进行嵌套，可以参考文档：如何将H5工程转为小程序。
  您可以通过FIDE进行代码的编辑调试，直到您的小程序在FIDE上表现正常，包括UI展示和业务逻辑。开发调试可以参考小程序开发文档：小程序开发

步骤四 真机预览#

  点击FIDE顶部菜单栏的预览按钮，FIDE经过编译后，会弹出小程序的预览二维码。
  打开FinClip App，选择企业端登录，登录步骤一注册的账号。登录成功后，点击app右上角的扫一扫按钮，扫描小程序预览二维码，即可在真机上查看小程序真机运行的效果。
  真机预览可参考开发文档：FIDE功介绍

步骤五 上架发布小程序#

  1. 从FIDE中导出小程序代码包，此时我们得到一个zip包，这个代码包是经过编译并且加密混淆过的，您无需担心源代码安全问题。
  2. 我们再次打开小程序开放平台，进入到我们刚刚创建好的小程序，点击进入详情，点击顶部的代码包管理，上传代码包，填写相关信息即可，我们的小程序代码包是从IDE中导出的，所以我们选择无需编译，即可完成小程序代码包的上传。
  3. 上传完成后，我们可以添加体验版. 审核版本。添加审核版本后，如果您是采用公有云方案，直接在小程序管理菜单下，小程序上架审核处查看到对应的小程序。
  4. 如果您是私有云的方案，则需要登录运营端，进入小程序管理-小程序上架审核页面，查看待审核的小程序。点击小程序详情，您可以选择通过或者是拒绝小程序审核。审核通过后，您可以在我的小程序中，进行小程序的上架和发布。
  小程序的上架文档参考：小程序上下架

步骤六 创建应用#

  到这个步骤，恭喜您已经将小程序开放编译调试. 上架. 发布的流程都已经完成。接下来您需要创建一个移动应用。
  1. 首先我们通过移动应用开发工具（Android Studio或者是Xcode）创建一个应用或者是打开现有的应用，获取到应用的 BundleID（移动应用的唯一标识）。
  2. 再次打开小程序开放平台，进入我们应用管理模块，点击新增合作应用按钮，填写应用名称和所属企业。
  3. 创建应用后，点击右侧的关联小程序按钮，关联我们的刚刚上架的小程序即可。再次点击对应的应用，点击添加BundleID按钮，添加我们的应用BundleID即可。
  通过此步骤，您将获得以下信息：SDK Key. SDK Secret. apiServer. apmServer. apiPrefix信息，该信息将用于移动应用集成SDK初始化时的参数。创建应用可以参考对应的文档：关联移动应用

步骤七 集成SDK#

  凡泰目前支持 iOS SDK . Android SDK. Flutter . React Native. Windows SDK 的集成，相关的集成文档如下：

  1. iOS SDK集成文档：iOS SDK集成
  2. Android SDK集成文档：Android SDK集成
  3. Flutter 集成文档：Flutter 集成
  4. React Native集成文档：React Native集成
  5. Windows SDK 集成文档：Windows SDK 集成

  集成SDK后，编译代码通过，采用步骤六中的 SDK Key. SDK Secret. apiServer. apmServer. apiPrefix信息进行SDK的初始化。调用SDK初始化的接口，会有初始化成功和失败的回调，您可以通过该回调，判断是否初始化SDK成功。

步骤八 打开小程序#

  调用SDK打开小程序的接口，将步骤五上架得到的AppId传入SDK接口，即可打开小程序。

  1. 安卓打开小程序的方式：打开小程序
  2. iOS 打开小程序的方式：打开小程序

  恭喜您！到此步骤之后，一个小程序已经能够完整的运行在您的App上了，接下来您就可以继续您的其他功能的验证。至此POC步骤流程结束。
